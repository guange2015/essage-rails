essage-rails
=========

[essage] for rails.

> a more elegant way to show message / tips.

Install 
----

`gem 'essage-rails'`


Usage
-----

1. add essage.js to bottom of your applicaton.html.erb

```
</body>
<%= javascript_include_tag "essage" %>
```

2. add css to your application.css

`*= require essage`

[essage]:https://github.com/sofish/essage
